## Technologies used ##

* Node.js
* TypeScript

## Configuration ##

npm `v16.17.0`

## Installation ##

Build project with node   
```bash
npm install
```

## Usage ##

Run app.js with nodemon   
```bash
npm start
```

Test project with Jest
```bash
npm test
```

Compile project with Typescript once  
```bash
npm build
```  

Compile project with Typescript and watch changes 
```bash
npm dev
```
